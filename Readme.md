# Read Carefully

## Setup Instructions

### 1
**windows**:
```sh
python -m venv venv
venv\Scripts\activate
```

**linux**:
```sh
python3 -m venv venv
source venv/bin/activate
```

### 2
```sh
pip install -r requirements.txt
```

### 3
```
python app.py
```
